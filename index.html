As ferramentas e tecnologias utilizadas para otimização de sites consistem em analisar os gostos e demandas dos usuários para assim estar entre os mais acessados nas buscas realizadas por qualquer pessoa.


A ausência de informações a respeito dos usuários que visitam o website ou compõem as mídias sociais proporcionam desvantagem competitiva no mercado.

Problema

O problema abordado nesta pesquisa visa identificar a dificuldade que as empresas têm para conhecer mais sobre seus usuários/clientes, ou seja, como essas empresas fariam para saber em qual serviço ou produto poderiam investir mais ou em qual deveriam deixar de dar uma certa atenção. Qual público mais se interessa pelos seus trabalhos ou descobrir qual a tendência do público para emplacar um projeto futuro de sucesso.

Hipótese

O trabalho levanta a hipótese de que traçar o perfil de seus usuários atribui vantagem competitiva a organizações. Este trabalho levanta a hipótese de que fazendo-se processos de captura/obtenção de dados sobre o público seria uma forma de obter vantagem competitiva no mercado. H0(hipótese nula) -> A vantagem competitiva no mercado independe de conhecer seus usuários. H1(hipótese alternativa) -> A vantagem competitiva no mercado também depende de conhecer seus usuários.

Objeto e Objetivo

O objeto desta pesquisa é a obtenção dos dados do clientes das organizações que utilizam a web como referência do seu trabalho; e o objetIvo, dar auxílio na tomada de decisões, quanto ao que fazer para aumentar o número de visitantes quanto consequente impacto dos dados obtidos na tomada de decisões no ambiente corporativo de modo que diferencie a empresa no mercado de trabalho.




SEO

SEO significa Search Engine Optimization, em português, otimização do mecanismo de pesquisa. Nada mais é do que um conjunto de métodos que visa a otimização de websites.



Técnicas de SEO

Para etiquetar o conteúdo de um website são utilizadas meta tags, elas servem para exibir para os buscadores qual é o conteúdo de determinado website. As meta tags são inseridas dentro de um conjunto de código HTML da página entre as tags 'head' e '/head'.

Google Analytics

O Google Analytics é um serviço oferecido pela gigante empresa Google, gratuito, utilizado para acompanhar estatísticas de visitação em um website. Esse sistema fornece um código Javascript que é inserido na página desejada, a partir daí este código envia dados ao Analytics e este repassa as informações ao dono. Em um website, o Analytics possui capacidade de identificar a taxa de exibição e hit de uma página, localização geográfica dos visitantes, de que modo acessaram determinada página (através de quais links, sites, buscadores, diretamente pela URL ou patrocínio), sistema operacional utilizado, navegador, versão do flash, Java, resolução de tela etc. Todos esses dados podem ser agrupados em períodos: diários, semanais, mensais ou anuais. Portanto, além de uma ferramenta poderosíssima de monitoramento de tráfego, o Google Analytics também funciona como uma excelente ferramenta para tomada de decisões em negócios Web. Apesar dele apontar as informações que precisam ser corrigidas, ele precisa de webmasters para efetuar as devidas manutenções nos sites.
